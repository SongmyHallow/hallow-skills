# 游戏分析维度库

分析时参考，不必面面俱到。根据目标游戏选择 4-5 个最相关的维度深入展开。

## 1. 核心玩法

- **核心循环**：玩家在 1 分钟 / 10 分钟 / 1 小时内分别做什么？
- **操作手感**：输入延迟、反馈感、镜头控制、动作流畅度
- **难度设计**：难度曲线、死亡惩罚、新手引导、"易上手难精通"做得怎样
- **机制创新**：有什么其他游戏没做过的东西？做成了吗？
- **玩法多样性**：流派/套路/配装有多少种？每种都能玩吗？还是只有一两个最优解？

## 2. 叙事与世界观

- **叙事结构**：线性/开放/碎片化叙事？剧情推进方式？
- **世界观构建**：世界观的原创性、内在一致性、细节密度
- **角色塑造**：主要角色的动机、成长、台词功底
- **叙事与玩法的关系**：剧情和玩法是协同还是割裂？（"我忙着打架根本没看剧情说了啥"）
- **情感共鸣**：有没有让你"破防"的瞬间？

## 3. 美术与视听

- **美术风格**：写实/卡通/像素/手绘？风格统一性
- **技术实现**：光影、材质、粒子效果、角色动画
- **UI/UX**：界面信息密度的合理性、交互流畅度
- **配乐**：BGM 质量、音效设计、配音（如有）
- **艺术表达**：有没有超越"好看"的审美追求？

## 4. 成长与经济

- **角色养成**：等级/技能/装备系统的深度和肝度
- **资源循环**：玩家如何获取资源、消耗资源？循环闭得漂亮吗？
- **付费设计**：买断/订阅/战斗通行证/抽卡/皮肤——怎么收钱？收得让人舒服吗？
- **经济平衡**：游戏内通货膨胀了吗？新玩家还追得上吗？

## 5. 技术表现

- **引擎与优化**：用的什么引擎？优化水平如何？
- **稳定性**：帧率、Bug频率、服务器质量
- **加载时间**：影响游戏体验吗？
- **平台适配**：PC/主机/移动端的适配质量
- **可访问性**：色盲模式、难度选项、字幕等辅助功能

## 6. 内容与运营（服务型游戏专用）

- **更新节奏**：多久更新一次？一次更多少？
- **内容质量**：更新是在加深度还是加宽度？新内容好不好玩？
- **赛季/活动设计**：赛季主题、限时活动、奖励结构
- **开发者沟通**：开发者怎么和玩家交流？透明吗？画饼吗？
- **平衡调整**：Buff/Nerf 频率和合理性，meta 多样性

## 7. 社群与生态

- **玩家社群**：在哪讨论？氛围怎样？核心玩家和二创玩家比例？
- **创作者生态**：mod 支持、同人创作、直播/视频内容
- **电竞/竞技**（如适用）：赛事体系、观赏性、天梯设计
- **速通社群**（如适用）：速通热度、技巧深度
- **留存模式**：玩家为什么反复回来？社交绑定、刷刷刷快感、还是 FOMO？

## 8. 历史定位

- **品类位置**：在这个品类里，它是开创者、集大成者、还是挑战者？
- **行业影响**：它的成功/失败改变了什么？有没有后来者模仿？
- **文化意义**：它有没有超越游戏本身，成为一种文化符号？
- **代际对比**：和系列前作、或同系列其他作品比，进步还是退步？

## 维度选择指南

| 游戏类型 | 重点维度 |
|---------|---------|
| 动作/ARPG | 1 核心玩法, 2 叙事, 3 美术, 5 技术 |
| 服务型/MMO | 1 核心玩法, 4 经济, 6 内容运营, 7 社群 |
| 独立游戏 | 1 核心玩法, 2 叙事, 3 美术, 8 历史定位 |
| 竞技/电竞 | 1 核心玩法, 5 技术, 7 社群 |
| 手游/F2P | 4 经济, 6 内容运营, 7 社群 |
| 叙事/步行模拟 | 2 叙事, 3 美术, 8 历史定位 |
