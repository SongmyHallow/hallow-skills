# Hallow Skills

我的个人 AI skill 库，用于 Claude Code 和 Gemini CLI 的自定义技能。

## 目录结构

```
hallow-skills/
├── claude/        # Claude Code 技能
├── gemini/        # Gemini CLI 技能
├── shared/        # 跨平台技能
└── tools/         # 辅助工具
```

## 使用方式

将 claude/ 目录中需要的 skill 复制到 Claude Code 的 skills 目录即可：

```bash
cp -r claude/<skill-name> ~/.claude/skills/
```
